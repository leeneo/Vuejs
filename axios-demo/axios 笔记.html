前端请求数据所用的网络模块有哪些

  1. 传统的 原生ajax ：XMLHttpRequest(XHR)

    • 配置和调用方式等混乱，编码难看，现实中很少使用

  2. jQuery ajax

    • 封装好的 ajax ，jQuery 时期非常好用，但到了移动端就不行了
    • jQuery重量级太大，放到移动端手机流量承受不住

  3. Vue-resource

    • vue 1.0 时期推出的
    • vue 2.0 出来后，vue resource 被抛弃

  4. Vue-axios

    • veu 2.0 时，Vue-resource 被遗弃后，vue-resource 的替代品就是 axios
