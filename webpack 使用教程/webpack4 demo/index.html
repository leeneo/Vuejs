使用 webpack 打包需要安装 webpack-cli

webpack4 打包js文件 有多种方式

  • webpack ./src/index.js -o ./dist/ 会在dist目录下自动生成 main.js
  • npx webpack
  • npx webpack --config webpack.config.js

添加 webpack.config.js 配置文件，可以简化很多操作，比如指定生成后的 js 文件名和目录

通过 npm scripts 方式运行 webpack

在package.json 加入 
        "scripts": {
            "test": "echo \"Error: no test specified\" && exit 1",
            "build": "webpack"
          },

之后通过 npm run build 命令，进行 webpack 打包

而且经过 scripts 配置过的 build 会优先运行本地的webpack

webpack4 相关

  • webpack-dev-server : webpack4 对应 webpack-dev-server@3.x.x 版本，dev server 超过这个版本会报错
  • webpack-dev-server : webpack-dev-server@3.x.x 对应 webpack cli3 版本，webpack cli 超过这个版本， npm run serve 时会报错
  • npm run "serve": "webpack-dev-server --open --config ./build/dev.config.js"
  • 参数说明：--open 项目启动后自动打开首页
