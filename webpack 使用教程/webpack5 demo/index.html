使用 webpack 打包需要安装 webpack-cli

webpack5 打包js文件方式

  • npx webpack
  • npx webpack --config webpack.config.js
  • npm run build

添加 webpack.config.js 配置文件，可以简化很多操作，比如指定生成后的 js 文件名和目录

通过 npm scripts 方式运行 webpack

在package.json 加入 
        "scripts": {
            "test": "echo \"Error: no test specified\" && exit 1",
            "build": "webpack"
          },

之后通过 npm run build 命令，进行 webpack 打包

而且经过 scripts 配置过的 build 会优先运行本地的webpack

mode: 'development',mode: 'production'

  • development: bundle.js 代码没有被压缩
  • production: bundle.js 代码是被压缩的
